| セレクタ                        | 説明                                             | 例                 | 適用される要素           |
| ------------------------------- | ------------------------------------------------ | ------------------ | ------------------------ |
| **隣接セレクタ** (`A + B`)      | `A`の直後にある`B`にスタイルを適用する           | `h2 + p`           | 直後の`p`要素            |
| **間接セレクタ** (`A ~ B`)      | `A`の後に出現するすべての`B`にスタイルを適用する | `h2 ~ p`           | `A`の後のすべての`p`要素 |
| **子孫セレクタ** (`A B`)        | `A`の任意の子孫である`B`にスタイルを適用する     | `div p`            | `div`内のすべての`p`要素 |
| **子セレクタ** (`A > B`)        | `A`の直下の子要素である`B`にスタイルを適用する   | `div > p`          | 直下の`p`要素            |
| **グループ化セレクタ** (`A, B`) | `A`と`B`両方にスタイルを適用する                 | `h1, h2`           | `h1`と`h2`               |
| **ユニバーサルセレクタ** (`*`)  | すべての要素にスタイルを適用する                 | `* { margin: 0; }` | 全要素                   |
| **属性セレクタ** (`[attr]`)     | 特定の属性を持つ要素にスタイルを適用する         | `[type="text"]`    | `type="text"`の要素      |

### 各セレクタの例

1. **隣接セレクタ (`A + B`)**

   ```css
   h2 + p {
     color: red;
   }
   ```

   `<h2>`の直後にある`<p>`要素が赤くなります。

2. **間接セレクタ (`A ~ B`)**

   ```css
   h2 ~ p {
     color: green;
   }
   ```

   `<h2>`の後に出現するすべての`<p>`要素が緑になります。

3. **子孫セレクタ (`A B`)**

   ```css
   div p {
     font-size: 18px;
   }
   ```

   `div`内のすべての`p`要素に適用されます。

4. **子セレクタ (`A > B`)**
   ```css
   div > p {
     font-size: 20px;
   }
   ```
   `div`の直下にある`p`要素にのみ適用されます。

## display関連

`display`プロパティは、CSSにおいて要素のレイアウト方法を指定する重要なプロパティです。主に以下のような違いや種類があります。

### 1. `inline`（インライン）

- **特徴**: 要素はテキストと同じ行に配置されます。幅や高さを設定できない（または無視される）。
- **例**: `<span>`, `<a>`, `<em>`など
- **使いどころ**: テキストの一部を装飾したり、リンクを追加する場合など。
- **挙動**:

  - 幅・高さが無視される。
  - 隣接する要素と同じ行に並ぶ。

  ```css
  .inline-element {
    display: inline;
  }
  ```

### 2. `block`（ブロック）

- **特徴**: 要素は新しい行に配置され、デフォルトで100%の幅を持ちます。幅や高さを自由に指定できます。
- **例**: `<div>`, `<p>`, `<h1>` など
- **使いどころ**: レイアウトの枠組みを作るときや、段落、セクションを区切る場合など。
- **挙動**:

  - 要素の前後に改行が入り、他の要素とは別の行になる。
  - 幅・高さを自由に設定できる。

  ```css
  .block-element {
    display: block;
  }
  ```

### 3. `inline-block`（インラインブロック）

- **特徴**: インライン要素のように同じ行に並びますが、ブロック要素のように幅や高さを指定できます。
- **使いどころ**: インラインで配置したいけど、サイズ調整が必要なボタンや画像など。
- **挙動**:

  - 他のインライン要素と同じ行に配置される。
  - 幅や高さが設定できる。

  ```css
  .inline-block-element {
    display: inline-block;
  }
  ```

### 4. `flex`（フレックス）

- **特徴**: 柔軟なレイアウトを作るためのモダンなレイアウトシステム。要素を並べる際、子要素の配置やサイズ調整が簡単にできる。
- **使いどころ**: レスポンシブなレイアウトや、要素を横並びにする場合に便利。
- **挙動**:

  - フレックスコンテナ内の子要素が自由に伸縮する。
  - 配置方法や並び順を調整可能。

  ```css
  .flex-container {
    display: flex;
  }
  ```

### 5. `grid`（グリッド）

- **特徴**: 2次元のレイアウトを簡単に管理できるレイアウトシステム。複雑なページレイアウトに適している。
- **使いどころ**: 大規模なレイアウトや複雑なページ構成を持つウェブページで使用。
- **挙動**:

  - グリッドコンテナ内で行と列を使って子要素を配置できる。

  ```css
  .grid-container {
    display: grid;
  }
  ```

### 6. `none`

- **特徴**: 要素を画面に表示しない（存在しないものとして扱われる）。
- **使いどころ**: 要素を非表示にするが、DOM上では残したい場合。
- **挙動**:

  - 要素は画面から消える。
  - 画面上のスペースを占めない。

  ```css
  .hidden-element {
    display: none;
  }
  ```

### その他の種類

- `table`, `inline-table`: 表のようなレイアウトを指定します。
- `list-item`: リスト項目の表示形式（デフォルトでリストマーカーがつく）。
- `run-in`: 見出しのように振る舞うが、次のブロック要素と連続して表示されることもある。
